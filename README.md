## About

This repository is a long-term archive of my accepted NeetCode submissions. Every solution is pushed here automatically by the NeetCode GitHub Sync the moment it is accepted on the platform.

NeetCode and LeetCode are linked, so every problem solved here counts towards both platforms. I follow the NeetCode 250 roadmap in order from the top, working through each topic section before moving on to the next. The goal is at least one problem solved every single day.

---

## Structure

Solutions are organised by topic folder then problem ID. Each submission is a separate file. If a problem is reattempted, each attempt is stored in order.

```
<topic-folder>/
  <problem-id>/
    submission-0.<ext>
    submission-1.<ext>
```

**Example:**

```
Data Structures & Algorithms/two-integer-sum/submission-0.py
Data Structures & Algorithms/binary-search/submission-0.java
Data Structures & Algorithms/valid-anagram/submission-0.cpp
Python For Beginners/python-hello-world/submission-0.py
```

## Sync Settings

Managed at [neetcode.io/profile/github](https://neetcode.io/profile/github).

- **Auto-commit** pushes every accepted submission automatically
- **Status filter** controls whether all submissions or accepted only are synced
- **Bulk Sync** pushes all past solutions at once if needed
